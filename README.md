# Jungle

A mini e-commerce application built with Rails 4.2 for purposes of teaching Rails by example.


## Project Setup

1. Fork & Clone
2. Run `bundle install` to install dependencies
3. Run `bin/rails s -b 0.0.0.0` to start the server
